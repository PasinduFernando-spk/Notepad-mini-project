#Mini Notepad

A Sample Java Swing application that functions a a lightweight notepad.you can open text files ,edit and save your work.

## setup instructions 

1. download the repository 
https://github.com/PasinduFernando-spk/Notepad-mini-project.git

2.Conpile the program 
you should have Java installed 

3.Run the program 
 java Main


##features

-Open text files
-Save files
-Exit from file menu
-Scrollable text area

##notes

-No database is required for this.
-The program reads and write plain text files only.
